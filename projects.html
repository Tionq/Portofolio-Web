KONVESI MATA UANG (IDR, USD, JPY) C++

Proyek ini merupakan program konversi mata uang yang dikembangkan menggunakan bahasa pemrograman C++. Aplikasi ini memungkinkan pengguna untuk mengonversi nilai mata uang antara Rupiah (IDR), Dolar Amerika (USD), dan Yen Jepang (JPY) secara dua arah.

Fitur Utama

  • Input dinamis untuk menentukan mata uang awal dan tujuan.
  • Penghitungan kurs mata uang secara akurat berdasarkan nilai tukar yang ditentukan.
  • Tampilan hasil yang informatif, termasuk penjelasan singkat tentang proses konversi.
  • Struktur program menggunakan kombinasi kondisi if-else untuk alur logika yang efisien.

    • Proyek ini menunjukkan kemampuan saya dalam pemrograman logika, manipulasi data, dan penerapan konsep-konsep dasar pengembangan perangkat lunak.

    Abstract Data Type: SIM

    Program ini dirancang menggunakan bahasa C++ untuk menampilkan informasi identitas dari kartu tertentu, seperti SIM (Surat Izin Mengemudi) dan KTP (Kartu Tanda Penduduk). Pengguna dapat memasukkan data identitas, seperti nama, nomor ID, alamat, jenis kelamin, dan informasi tambahan lainnya. Program akan memproses data ini dan menampilkannya kembali dalam format yang rapi dan mudah dibaca.

    Fitur Utama

  • Input data identitas
  • Penyimpanan data sementara
  • Output data identitas
  • Validasi input

    • Proyek ini menunjukkan kemampuan saya memahami lebih dalam mengenai apa itu ADT (Abstract Data Type).

    Membuat Artikel Ilmiah

    Proyek ini adalah artikel ilmiah yang membahas nilai-nilai keteladanan tokoh-tokoh dalam novel Misteri Pantai Mutiara karya Erlita Pratiwi. Artikel ini bertujuan untuk menganalisis karakter para tokoh dalam novel yang dapat dijadikan teladan, baik dalam aspek moral, kepemimpinan, maupun keberanian menghadapi tantangan.

    Poin Pembahasan Utama:

  • Analisis mendalam terhadap karakter utama dan sifat-sifat positif yang mereka tunjukkan.
  • Penekanan pada relevansi nilai-nilai tersebut terhadap pembentukan karakter generasi muda.
  • Penyusunan artikel mengikuti format ilmiah, lengkap dengan referensi dan analisis berbasis teori sastra.

    • Melalui proyek ini, saya menunjukkan kemampuan dalam analisis literatur, penulisan ilmiah, serta pemahaman terhadap pesan moral dalam karya sastra.
